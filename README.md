## Quick startv

- ### Install [Node.js](https://nodejs.org/)

    Check node version
  
    ```
    node -v
    ```

- ### Install [yarn](https://yarnpkg.com/)

    Node.js >=16.10

    ```
    corepack enable
    ```

    Node.js <16.10

    ```
    npm i -g corepack
    ```

    Check yarn version

    ```
    yarn -v
    ```

- ### Install [Git](https://git-scm.com/)

    Check git version

    ```
    git --version
    ```

- ### Clone this repository

    ```
    git clone https://github.com/champnc/chaoteaw-bot.git
    ```

- ### Change directory & run yarn install

    ```
    yarn install
    ```

- ### run yarn build

    ```
    yarn run build:run
    ```
